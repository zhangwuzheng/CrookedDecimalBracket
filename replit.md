# 藏境山水 - 极简风格移动端页面

## 项目概述
一个移动端优先的网页，展示藏区特产品牌。使用视频作为全屏背景，底部图标区域采用极简风格和半透明毛玻璃效果。

## 当前状态
- 视频背景：使用 attached_assets/92548_720p_1764129968331.mp4
- 图标区域：极简线性SVG图标，半透明背景，毛玻璃效果

## 项目结构
```
/
├── index.html          # 主页面（包含内联CSS和JS）
├── attached_assets/    # 资源文件夹
│   └── 92548_720p_1764129968331.mp4  # 视频背景
├── style.css           # 外部样式（目前未使用）
├── script.js           # 外部脚本（目前未使用）
└── replit.md           # 项目说明文档
```

## 技术实现
- 视频背景：使用HTML5 video标签，autoplay、loop、muted属性
- 半透明效果：backdrop-filter: blur(20px) + rgba背景色
- 图标：纯SVG线性图标，stroke风格
- 动画：slideUp入场动画

## 开发命令
- 启动服务器：`python -m http.server 5000 --bind 0.0.0.0`
